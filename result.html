Results of IISER Aptitude Test (IAT) 2025

The Results of IAT 2025 can be viewed in the Application Portal


The Application Portal link can be found at the bottom of this page.


Please read the information and instructions given below carefully.

Important Information:

Please read carefully the Information Brochure of IAT 2025. Click here for the English Version and click here for the Hindi Version of the Information Brochure.

Important Information Related to IAT 2025 Results:

  • The IAT 2025 results have been declared on June 24, 2025.
  • The candidates can view and download the rank card once they login in the Application Portal.
  • Candidate can login with the User ID and Password to access the application form. The User ID (Registration/Application Number) and Password were sent to the registered email ID and phone number during the registration process.
  • The candidates who have scored atleast ONE mark in the IAT 2025, have secured ranks.
  • Please note that, getting a rank does not guarantee offer/admission to an IISER.

Instructions:

  • Kindly use Edge Chromium or Mozilla Firefox (version 97 to 114) or Google Chrome (version 91 to 114) or Microsoft Edge (version 91 to 114) browser to edit the Application Form.
  • The Application Form is compatible with Android (version 4.0 and above) and iOS (version 9 and above).
  • It is recommended to view the Application Portal using Desktop/Laptop.
